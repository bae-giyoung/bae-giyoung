## 📌 대표 프로젝트 (Featured Projects)

### 🌊 AI 기반 지하수위 예측 대시보드

> KISTI 주관 『2025 DATA·AI 분석 경진대회』 과제로 진행한  
> AI 기반 지하수위 예측 결과 시각화 웹 대시보드 프로젝트입니다.

#### 프로젝트 개요

기상·수문 데이터를 기반으로 LSTM·Transformer 모델의 지하수위 예측 결과를  
Next.js 대시보드에서 시각화한 End-to-End 웹서비스입니다.

| 항목 | 내용 |
|------|------|
| 기간 | 2025.09 ~ 2025.11 |
| 팀 구성 | 3명: 데이터 / 백엔드 / 프론트엔드 |
| 역할 | 프론트엔드 개발 · 팀장 |
| 주요 기술 | Next.js, React, TypeScript, Tailwind CSS, Spring Boot, FastAPI, MySQL |
| 핵심 키워드 | 데이터 시각화, API 연동, CORS, 세션 인증, 다중 서버 연동 |

#### 주요 성과

- KISTI 주관 2025 DATA·AI 분석 경진대회 본선 진출
- AI 예측 결과를 사용자가 확인할 수 있는 웹 대시보드 MVP 완성
- Next.js · Spring Boot · FastAPI를 연결한 End-to-End 서비스 흐름 검증
- Next.js rewrites 기반 프록시 구성으로 다중 서버 CORS 및 세션 쿠키 이슈 대응
- REST API 명세서 작성 및 프론트엔드-백엔드 연동 기준 정리
- 프로젝트 종료 후 Spring Boot 중심 API Gateway 구조로 확장·개선 방향 도출

#### 담당 역할

- Next.js 기반 지하수위 현황 · 예측 대시보드 UI/UX 설계 및 구현
- 지도, 차트, 표, 모달 등 데이터 시각화 컴포넌트 개발
- Spring Boot · FastAPI API 연동 및 예측 데이터 시각화 구현
- 외부 기상 API 수집 데이터와 모델 추론 결과가 UI까지 전달되는 흐름 통합 테스트
- 팀장으로서 기능 범위 조율, 일정 관리, 발표 및 문서화 방향 정리

### 🧑‍💻 개인 프론트엔드 포트폴리오 웹사이트

> 기술 스택, 프로젝트 경험, 개발 역량을 소개하기 위해  
> 기획·디자인·개발·배포를 1인으로 진행한 개인 포트폴리오 사이트입니다.

#### 프로젝트 개요

Next.js App Router 기반으로 제작한 프론트엔드 개발자 포트폴리오입니다.  
프로젝트 상세 페이지는 SSG로 사전 생성하고, GSAP·Framer Motion·Lenis를 활용해  
포트폴리오 성격에 맞는 인터랙션과 페이지 전환 경험을 구현했습니다.

| 항목 | 내용 |
|------|------|
| 기간 | 2026.04 ~ 2026.05 |
| 형태 | 1인 기획 · 디자인 · 개발 · 배포 |
| 주요 기술 | Next.js 16, React 19, TypeScript, Tailwind CSS v4, GSAP, Framer Motion, Lenis, Jotai |
| 배포 | Vercel |
| 핵심 키워드 | SSG, 인터랙션, 이미지 최적화, 자동 배포, 포트폴리오 구조화 |

#### 주요 성과

- GitHub–Vercel 연동으로 자동 배포 흐름 구성
- SSG 기반 프로젝트 상세 페이지 사전 생성으로 CDN 중심 정적 응답 구조 구현
- Node.js + sharp 기반 WebP 자동 압축 파이프라인 구현
- 이미지 에셋 10,528 KB → 906 KB로 약 91.4% 용량 절감
- Jotai 기반 테마, 모달, 토스트, 인트로 상태 관리
- GSAP · Framer Motion · Lenis 기반 페이지 전환 및 스크롤 인터랙션 구현

#### 담당 역할

- Next.js App Router 기반 사이트 기획·디자인·개발·배포 전담
- `generateStaticParams`를 활용한 프로젝트 상세 페이지 정적 생성 구현
- 프로젝트 데이터 싱글 소스 구조 설계
- Node.js + sharp 기반 이미지 최적화 자동화 스크립트 구현
- GSAP 커튼 인트로, 페이지 전환, 커서 효과 등 인터랙션 구현
